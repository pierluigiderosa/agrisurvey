Installa Geopaparazzi

Geopaparazzi è uno strumento sviluppato per eseguire rilievi digitali in campo. Anche se il fine è fare dei rilievi, l'app permette di utilizzare propri dati vettoriali (spatialite) e mappe di sfondo.

Installare l'app su un dispositivo android e avviare l'app per fargli generare tutte le cartelle di supporto.

Installa!

Copia i file nel tuo sistema

Devi copiare 2 file nella cartella Geopaparazzi presente nella memoria principale del tuo dispositivo. Una volta avviata l'app per la prima volta.

Copiare il file per rilievo.json nella cartella Geopaparazzi l'app su un dispositivo android e avviare l'app per fargli generare tutte le cartelle di supporto.

Rilievo.json

Copia i file per un utilizzo online e offline delle mappe

Devi copiare i file nella cartella Geopaparazzi presente nella memoria principale del tuo dispositivo. Una volta avviata l'app per la prima volta è possibile aggiungere strati informativi offline e online.

Copiare il file per OrtofotoRealvista-mapurl nella cartella Geopaparazzi l'app su un dispositivo android e avviare l'app per fargli generare tutte le cartelle di supporto.

Ortofoto.mapurl
